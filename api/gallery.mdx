## Endpoint

```
GET /api/gallery
```

No authentication required. Returns a static JSON array from `src/app/gallery.json`.

## Updating the gallery

To add a new entry:

1. Run an audit and get the `blobId`
2. Add an entry to `src/app/gallery.json`
3. Run `npx tsx scripts/gallery-audits.ts` to verify all blobs are fetchable (HTTP 200)

The gallery is displayed on the landing page (`/`) as clickable cards that link to `/audit/<id>` for live-fetched reports.

<Note>
  The gallery endpoint is read-only and statically served. It does not reflect real-time audit results — only the curated entries in `gallery.json`.
</Note>
